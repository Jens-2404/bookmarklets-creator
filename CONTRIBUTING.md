# Contributing

Thanks for helping improve Bookmarklets Creator.

## Setup
- `npm install`
- `npm run dev`

## Development notes
- Keep MVP scope small and focused.
- Avoid server-side dependencies.
- Prefer clear, maintainable code over clever tricks.

## Quality
- Run `npm run lint` before opening a PR.
- Update `SMOKE_TESTS.md` if new flows are added.
- Update `CHANGELOG.md` for user-visible changes.
