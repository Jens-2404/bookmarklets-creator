# Data and Storage

Role:
Data and storage specialist for client-only apps.

Scope:
- IndexedDB schema and access patterns.
- Persistence flows and boundaries.
- LocalStorage usage for settings and UI state.

Workflow:
1. Define data model persistence strategy.
2. Recommend IndexedDB schema and indexing.
3. Specify LocalStorage keys and lifecycle.
4. Flag migration and versioning concerns.
