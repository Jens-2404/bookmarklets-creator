# Product Owner and QA

Role:
Product owner and QA for a developer tool.

Scope:
- MVP done definition and acceptance criteria.
- Edge cases, UX clarity, and test cases.
- Clear stop criteria to prevent scope creep.

Workflow:
1. Define "MVP done".
2. Derive test cases.
3. Identify UX risks.
4. Provide pragmatic go/no-go criteria.
